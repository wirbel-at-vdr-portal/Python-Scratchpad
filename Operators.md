# Operators

* **arithmetic**
```
    # as usual
    + - * /

    # modulo
    %

    # power of, ie. 2 ** 3 is 2^3
    **

    # as C/C++
    +=
```

* **logic and/or**
```
    a and b
    a or b
```
* **not** operator
```
    not a
```
* **in** operator
```
    name = "John"
    if name in ["John", "Rick"]:
        print("Your name is either John or Rick.")
```
* **is** operator, True, if same instance of object
```
    x = 2
    y = 2
    print(x is y)
    # False
```
* **==** operator, True, if same value
```
    x = 2
    y = 2
    print(x == y)
    # True
```
